Tipe Boolean
--------------------------------------------------
- Tipe boolean merepresentasikan nilai True atau False
- Tipe boolean akan membandingkan 2 atau lebih nilai atau variable
- contoh : tipe_boolean.py
           tipe_boolean2.py

Fungsi bool()
------------------------------
- Fungsi bool() digunakan untuk mengevaluasi sebuah nilai. Jika ada akan bernilai Frue, dan jika kosong bernilai False.
- contoh : fungsi_bool.py

Fungsi Return Boolean
- Anda dapat membuat sebuah fungsi yang mengembalikan nilai boolean.
- contoh : fungsi_return_bool.py
